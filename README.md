# Business-Performance-Dashboard-Excel

## Overview
This project is an interactive Excel dashboard built using Pivot Tables and Pivot Charts to analyze sales data from an offline store.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization

## Dashboard Features
- Total Profit by Payment Method
- Total Profit by Product Line
- Revenue, Profit & Expense Analysis
- Transaction Analysis by Region
- Average Expense by Department

## Skills Demonstrated
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Business Data Analysis
